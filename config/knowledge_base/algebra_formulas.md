# Algebra Formulas & Identities

## Quadratic Equations

### Quadratic Formula
For ax² + bx + c = 0:
x = (-b ± √(b² - 4ac)) / (2a)

### Discriminant
D = b² - 4ac
- D > 0: Two distinct real roots
- D = 0: One repeated real root (equal roots)
- D < 0: Two complex conjugate roots

### Sum and Product of Roots
If α and β are roots of ax² + bx + c = 0:
- α + β = -b/a
- α × β = c/a

### Vieta's Formulas (General)
For polynomial aₙxⁿ + aₙ₋₁xⁿ⁻¹ + ... + a₁x + a₀ = 0:
- Sum of roots = -aₙ₋₁/aₙ
- Product of roots = (-1)ⁿ × a₀/aₙ

## Factoring Techniques

### Common Factoring Patterns
- a² - b² = (a + b)(a - b)    [Difference of squares]
- a³ + b³ = (a + b)(a² - ab + b²)    [Sum of cubes]
- a³ - b³ = (a - b)(a² + ab + b²)    [Difference of cubes]
- a² + 2ab + b² = (a + b)²    [Perfect square trinomial]
- a² - 2ab + b² = (a - b)²    [Perfect square trinomial]

### Completing the Square
ax² + bx + c = a[(x + b/2a)² - (b² - 4ac)/4a²]

## Inequalities

### Properties
- If a > b and c > 0, then ac > bc
- If a > b and c < 0, then ac < bc (inequality flips)
- If a > b > 0, then 1/a < 1/b
- AM ≥ GM: (a + b)/2 ≥ √(ab) for a, b ≥ 0

### Absolute Value
- |x| < a ⟹ -a < x < a
- |x| > a ⟹ x < -a or x > a
- |a + b| ≤ |a| + |b|    [Triangle inequality]

## Sequences & Series

### Arithmetic Progression (AP)
- nth term: aₙ = a + (n-1)d
- Sum of n terms: Sₙ = n/2 × [2a + (n-1)d] = n/2 × (a + l)
- Common difference: d = aₙ₊₁ - aₙ

### Geometric Progression (GP)
- nth term: aₙ = ar^(n-1)
- Sum of n terms: Sₙ = a(rⁿ - 1)/(r - 1) for r ≠ 1
- Sum to infinity: S∞ = a/(1 - r) for |r| < 1

## Logarithms

### Properties
- log_a(xy) = log_a(x) + log_a(y)
- log_a(x/y) = log_a(x) - log_a(y)
- log_a(x^n) = n × log_a(x)
- log_a(b) = log_c(b) / log_c(a)    [Change of base]
- log_a(a) = 1
- log_a(1) = 0

## Binomial Theorem

### Binomial Expansion
(x + y)ⁿ = Σ C(n,r) × x^(n-r) × y^r for r = 0 to n

### General Term
T(r+1) = C(n,r) × x^(n-r) × y^r

### Properties of Binomial Coefficients
- C(n,0) + C(n,1) + ... + C(n,n) = 2ⁿ
- C(n,0) - C(n,1) + C(n,2) - ... = 0
- C(n,r) = C(n, n-r)

## Complex Numbers

### Forms
- Rectangular: z = a + bi
- Polar: z = r(cosθ + i sinθ) = r × e^(iθ)

### Properties
- |z| = √(a² + b²)
- Conjugate: z̄ = a - bi
- z × z̄ = |z|²

### De Moivre's Theorem
(cosθ + i sinθ)ⁿ = cos(nθ) + i sin(nθ)
